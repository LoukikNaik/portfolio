# Personal Portfolio Website

A modern, responsive portfolio website built with React and Tailwind CSS.

## Features

- Responsive design that works on all devices
- Modern and clean UI
- Smooth scrolling navigation
- Project showcase section
- Skills display with icons
- Contact form
- Social media integration

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/portfolio-website.git
```

2. Navigate to the project directory
```bash
cd portfolio-website
```

3. Install dependencies
```bash
npm install
# or
yarn install
```

4. Start the development server
```bash
npm start
# or
yarn start
```

5. Open [http://localhost:3000](http://localhost:3000) to view it in your browser

## Customization

1. Update the personal information in the components:
   - Edit `Hero.js` to update your name and title
   - Modify `About.js` with your background information
   - Update `Projects.js` with your project details
   - Customize `Skills.js` with your technical skills
   - Update social media links in `Footer.js`

2. Replace placeholder images with your own images in the projects section

3. Add your own color scheme by modifying the Tailwind configuration

## Building for Production

To create a production build:

```bash
npm run build
# or
yarn build
```

The build files will be created in the `build` folder.
